# Webservice REST API Reference

Static reference generated from the CA Biositing webservice OpenAPI spec.
The spec file (`openapi.json`) is committed to the repository and can be refreshed with:

```bash
pixi run -e webservice generate-openapi
```

For a live, interactive version start the webservice and visit
[http://localhost:8000/docs](http://localhost:8000/docs):

```bash
pixi run start-webservice
```
